  • General Purpose Processor

    Winter 2021/22

    Final data path drawn by a partner.

    As a final project for my computer architecture course, three other students and I created a simulated general purpose stack processor in Xilinx. The processor takes 16-bit instructions with 17 different opcodes to execute different operations. Our final processor is able to execute euclids algorithm as well as most common operations. It is a multi-cycle processor with three to four stages for each instruction, greatly increasing its efficiency. I also created an assembler for this processor turning our assembly code into machine language. The Github repository for this project as well as our final report can be found by clicking the tile. (Coming soon)

  • Editor AVL Tree Data Structure

    Fall 2021

    AVL tree with balance factors.

    As a final project for my data structures and algorithms class another student and I created a modified AVL tree data structure intended to be efficient for a word editor. On top of its self-balancing nature, we also implemented a "rank" system that would allow the program to find a nodes position in the string the tree represents. This allowed for insertion and deletion at any position in O(log(n)) time. The Github repository for this project can be found by clicking the tile. (Coming soon)

  • Genetic Algorithm

    Spring 2021

    A screenshot of the UI of our project during an evolution simulation.

    As a final project for my object oriented design class another student and I decided to make a simple evolutionary algorithm. Our program allows you to modify factors such as mutation rate, population size, generation count, and many more. Our algorithm also has support for multiple types of selection like truncation or roulette wheel. You are also able to the diversity and average fitness throughout the evolutionary cycles as well as the chromosome of the agent with the best fitness.
